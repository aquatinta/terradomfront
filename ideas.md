## Идея 1: Промышленный минимализм (Industrial Brutalism Lite)

**Design Movement:** Новый брутализм + промышленный минимализм
**Core Principles:**
- Жёсткие геометрические формы, острые углы, никаких скруглений
- Контрастная типографика: жирный гротеск против тонкого засечного
- Монохромная база с одним ярким акцентом (оранжевый #E85D04)
- Структурные линии как декоративные элементы

**Color Philosophy:** Тёмно-серый (#1A1A1A) + белый + строительный оранжевый. Отсылает к стройплощадке, но в высокотехнологичной интерпретации.
**Layout Paradigm:** Асимметричная сетка с жёсткими колонками. Текст и изображения не выровнены по центру.
**Signature Elements:** Горизонтальные линии-разделители, нумерованные секции (01, 02, 03), жирные заголовки на весь экран
**Interaction Philosophy:** Резкие, мгновенные переходы без плавности
**Animation:** Slide-in по горизонтали, никаких fade-in
**Typography System:** Neue Haas Grotesk / Space Grotesk (заголовки) + Roboto Mono (акценты)

## Идея 2: Органический Земной (Organic Earth Tones)

**Design Movement:** Скандинавский органицизм + PropTech
**Core Principles:**
- Тёплые земляные тона, текстуры дерева и камня
- Плавные кривые, мягкие тени, ощущение уюта
- Баланс между технологичностью и природностью
- Большие воздушные секции с щедрым whitespace

**Color Philosophy:** Тёплый кремовый (#F5F0E8) + тёмный лесной зелёный (#2D4A3E) + золотистый акцент (#C9A84C). Ассоциация с землёй, деревом, домом.
**Layout Paradigm:** Широкие горизонтальные полосы с чередованием текста и изображений, плавные переходы между секциями
**Signature Elements:** Органические SVG-формы как разделители секций, иллюстрации домов в тёплых тонах, текстурные фоны
**Interaction Philosophy:** Плавные, почти медленные переходы — ощущение спокойствия и надёжности
**Animation:** Parallax-скролл, медленный fade-in снизу
**Typography System:** Playfair Display (заголовки) + Source Sans Pro (тело)
</text>

## Идея 3: Технологичный Тёмный (Dark Tech PropTech) ← ВЫБРАНА

**Design Movement:** Dark PropTech / Архитектурный Цифровой
**Core Principles:**
- Тёмный фон с элементами архитектурных чертежей как текстуры
- Акцентный цвет — насыщенный янтарный/золотой (#F59E0B) — как подсветка на чертеже
- Стеклянные карточки (glassmorphism) для секций партнёров
- Чёткая иерархия через контраст яркости

**Color Philosophy:** Глубокий графитовый (#0F1117) + тёмно-синий (#0D1B2A) + янтарный акцент (#F59E0B) + чистый белый для текста. Ощущение высокотехнологичного строительного проекта, архитектурного бюро.
**Layout Paradigm:** Полноэкранные секции с диагональными разделителями. Hero — разделён по диагонали: левая часть текст, правая — 3D-визуализация дома. Нестандартное расположение элементов.
**Signature Elements:** Тонкие линии-сетки (blueprint grid) как фоновая текстура, светящиеся акцентные линии, числовые метрики крупным шрифтом
**Interaction Philosophy:** Hover-эффекты с подсветкой, кнопки с анимированной рамкой
**Animation:** Плавный slide-up при скролле, пульсирующие акценты, анимированные счётчики цифр
**Typography System:** Manrope (заголовки, жирный) + Inter (тело текста, читаемость)
</text>

## Выбранный подход: Идея 3 — Технологичный Тёмный (Dark Tech PropTech)

Этот стиль наилучшим образом отражает позиционирование «Террадом» как технологической платформы с 3D/AR-конфигуратором. Тёмный фон с архитектурными текстурами создаёт ощущение профессионального инструмента, янтарный акцент — тепло и надёжность, а glassmorphism-карточки — современность.
